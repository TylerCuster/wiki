This is a toy wiki site constructed as a final project for Udacity's CS 253 course Web Development Fundamentals.

It's a demonstration of server-side Python deployed by Google App Engine using webapp2 handlers and Jinja2 templates. Data is stored through GAE's Datastore.

The code for the project is available at Github.

Some things to do:

  • Signup or login.
  • Create a new entry by clicking 'new entry' or typing the URL '/title' with your desired new entry title.
  • Edit an entry by clicking 'edit' on a page or typing the URL '/_edit/title' with your desired title.
  • View the edit history by clicking 'history' on a page or typing the URL '/_history/title' with your desired title.

Current issues:

  • HTML is not escaped in the page entries. I'm trusting 'users' not to mess with my site. This is sensible because the only 'user' is myself.
